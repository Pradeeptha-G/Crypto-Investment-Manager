# 🚀 Crypto Investment Manager

A Python-based system for cryptocurrency analysis, prediction, and portfolio optimization.

---

## 📌 Project Overview
Crypto Investment Manager helps users analyze crypto data, predict trends, and generate optimal investment strategies using rule-based logic and historical data.

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- Scikit-learn  
- Flask  
- SQLite  

---

## ✨ Features
- 📊 Data collection using CoinGecko API  
- 📈 EDA and visualization  
- ⚙️ Investment mix calculator  
- 🔍 Risk analysis and prediction  
- 📄 Report generation  
- 🔔 Alerts for major changes  

---

## 📂 Project Structure
- `data_collector.py` – Collects crypto data  
- `eda.py` – Performs analysis  
- `eda_visuals.py` – Visualization  
- `risk_based_mix_calculator.py` – Investment logic  
- `app.py` – Main application
